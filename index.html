DRIVER AND MONITORING FEATURES FOR THE 200/25/ZR

The following are a listing of a few selected driver and monitoring features which have been applied to my 1996 rover 200 and the 2004 MG-ZR 120 plus. A few applications will also deliver both audible, and visual vehicle information back to the driver.

COOLANT MONITORING ROVER 200/25/ZR

To install this feature the cooling system did not need to be drained,  off, just a small amount syphoned out while the tanks are swopped over. A coolant loss alert was then needed to be added.

SECURITY NAVIGATION SYSTEM ROVER 200
